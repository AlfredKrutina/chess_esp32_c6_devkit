\doxysubsection{Detailn√≠ popis}
ESP32-\/\+C6 Chess System -\/ Hlavicka Streaming Output Systemu. 

Nahradi pameti intenzivni budovani stringu primym streaming vystupem\+:
\begin{DoxyItemize}
\item Eliminuje potrebu velkych bufferu (usetri 2KB+ na velky vystup)
\item Snizuje fragmentaci pameti
\item Umoznuje real-\/time postupny vystup
\item Podporuje vice vystupnich cilu (UART, Web, Queue)
\end{DoxyItemize}

Alfred Krutina 
\end{DoxyAuthor}
\begin{DoxyVersion}{Verze}

\end{DoxyVersion}
\begin{DoxyDate}{Datum}

Streaming Output System umoznuje efektivni vypis velkych textu bez budovani velkych stringu v pameti. Posila data primo do vystupu, coz eliminuje potrebu velkych bufferu a zlepsuje performance.
