\doxysubsection{Detailní popis}
ESP32-\/\+C6 Chess System v2.\+4 -\/ Timer System implementace. 

Tato komponenta implementuje casovy system pro sachovou hru\+:
\begin{DoxyItemize}
\item Presne mereni casu s ESP32 timer API
\item Thread-\/safe operace s casem
\item Ruzne typy casovych kontrol
\item JSON API pro web rozhrani
\item NVS persistence nastaveni
\end{DoxyItemize}
