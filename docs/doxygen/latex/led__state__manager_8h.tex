\doxysubsection{Detailní popis}
LED State Manager -\/ Pokrocila sprava LED stavu a vrstev. 

Tento modul poskytuje komplexni spravu LED stavu\+:
\begin{DoxyItemize}
\item Vrstvovy system pro LED efekty
\item Prioritni spravce LED
\item Perzistentni LED stav
\item Optimalizovane aktualizace
\item Podpora pro kompozitni efekty
\end{DoxyItemize}

LED State Manager poskytuje pokrocily system pro spravu LED stavu. Umoznuje kombinovat vice efektu najednou pomoci vrstvoveho systemu, kde kazda vrstva ma svou prioritu a blending mode.

Vyhody\+:
\begin{DoxyItemize}
\item Vrstvovy kompoziting (sachovnice + efekty + GUI)
\item Prioritni systém s alpha blendingem
\item Optimalizovane dirty-\/pixel updates
\item Thread-\/safe pristup ke stavu
\item Perzistence mezi aktualizacemi 
\end{DoxyItemize}
