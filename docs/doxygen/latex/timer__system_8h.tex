\doxysubsection{Detailní popis}
ESP32-\/\+C6 Chess System v2.\+4 -\/ Timer System komponenta. 

Tato komponenta spravuje časový systém pro sachovou hru\+:
\begin{DoxyItemize}
\item Ruzne typy casovych kontrol (bullet, blitz, rapid, classical)
\item Presne mereni casu s ESP32 timer API
\item Thread-\/safe operace s casem
\item Integrace s game taskem
\item Web API pro ovladani casu
\end{DoxyItemize}

\begin{DoxyVersion}{Verze}

Timer system poskytuje kompletni casovou kontrolu pro sachovou hru. Podporuje standardni casove kontroly a vlastni nastaveni. Vsechny operace jsou thread-\/safe a optimalizovane pro ESP32.
