\doxysubsection{Detailní popis}
Visual Error System -\/ Vizualni indikace chyb a navod pro uzivatele. 

Tento modul poskytuje vizualni system pro zpracovani chyb\+:
\begin{DoxyItemize}
\item Indikace chyb pomoci LED
\item Textovy popis chyb
\item Uzivatelsky navod pro napravu
\item Prioritni system pro vice chyb soucasne
\item Integrace s ostatnimi vizualnimi systemy
\end{DoxyItemize}

Vizualni system pro zpracovani chyb poskytuje komplexni system pro zobrazovani chyb uzivateli. Kombinuje LED indikace s textovym popisem a poskytuje jasny navod jak chybu opravit.

Vyhody\+:
\begin{DoxyItemize}
\item Jasne LED indikace chyb
\item Preklady chyb do cestiny
\item Krok-\/po-\/kroku navod pro opravu
\item Automaticke zmizeni po oprave
\item Podpora pro vice chyb soucasne
