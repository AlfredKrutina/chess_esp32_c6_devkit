Enhanced Castling System pro ESP32 Sachovou Sachovnici. 

Tento modul poskytuje komplexni, robustni system pro rosadu s\+:
\begin{DoxyItemize}
\item Centralizovanou spravou stavu
\item Pokrocilym LED navodem a indikaci chyb
\item Inteligentnim error recovery
\item Timeout handlingem
\item Vizualnim navodem pro hrace
\end{DoxyItemize}
