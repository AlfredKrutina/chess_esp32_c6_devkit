Unified Animation Manager -\/ Jednotna sprava vsech LED animaci. 

Tento modul poskytuje jednoduche rozhrani pro vsechny LED animace\+:
\begin{DoxyItemize}
\item Centraliz ovana sprava animaci
\item Prioritni system pro konfliktni animace
\item Jednoduche API pro rychle pouziti
\item Podpora pro vice soucasne bezicich animaci
\item Plynule prechody mezi animacemi
\end{DoxyItemize}

Unified Animation Manager poskytuje centralizovany system pro spravu vsech LED animaci v systemu. Poskytuje prioritni system, podpora pro vice soucasne bezicich animaci a plynule prechody.

Vyhody\+:
\begin{DoxyItemize}
\item Jednoduche API\+: animation\+\_\+start(\+ANIM\+\_\+\+MOVE, from, to, duration)
\item Automaticke conflict resolution s prioritami
\item Podpora pro stackovani animaci (napr. check + move)
\item Plynule fade-\/in/fade-\/out prechody
\item Thread-\/safe pristup 
\end{DoxyItemize}
