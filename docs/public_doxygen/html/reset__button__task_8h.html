Detailní popis

Reset Button Task hlavicka pro ESP32-C6 sachovy projekt.

Tato hlavicka definuje rozhrani pro reset button task:

  • Inicializace reset button tasku a FreeRTOS komponent
  • Zpracovani reset tlacitka pro restart hry
  • Simulacni rezim bez hardware (pro development)
  • Integrace s game taskem pro reset
  • Jedno tlacitko pro reset cele hry
Autor
Alfred Krutina
Verze
2.0
Datum
2025-08-16

Tento task zpracovava reset tlacitko pro restart hry. Kdyz hrac stiskne reset tlacitko (GPIO15), hra se restartuje do vychoziho stavu. Task detekuje stisknuti a posila prikaz game tasku.
