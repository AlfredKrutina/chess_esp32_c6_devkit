Detailní popis

PRIME LED FUNKCE PRO GAME TASK - Zadny Queue Hell.

RESENI: Misto xQueueSend(led_command_queue) pouzivat prime LED volani

  • Zadny queue hell
  • Zadne timing problemy
  • Okamzite LED aktualizace
Autor
Alfred Krutina
Verze
2.5 - DIRECT SYSTEM
Datum
2025-09-02

Tento modul poskytuje prime LED funkce pro game task bez pouziti front. Eliminuje queue hell a timing problemy tim, ze vola LED funkce primo. Vsechny funkce jsou thread-safe diky pouziti mutexu v led_task.
