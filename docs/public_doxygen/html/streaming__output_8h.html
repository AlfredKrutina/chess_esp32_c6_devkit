DetailnÃ­ popis

ESP32-C6 Chess System - Hlavicka Streaming Output Systemu.

Nahradi pameti intenzivni budovani stringu primym streaming vystupem:

  • Eliminuje potrebu velkych bufferu (usetri 2KB+ na velky vystup)
  • Snizuje fragmentaci pameti
  • Umoznuje real-time postupny vystup
  • Podporuje vice vystupnich cilu (UART, Web, Queue)
Autor
Alfred Krutina
Verze
2.4
Datum
2025-01-27

Streaming Output System umoznuje efektivni vypis velkych textu bez budovani velkych stringu v pameti. Posila data primo do vystupu, coz eliminuje potrebu velkych bufferu a zlepsuje performance.

stream_board_row(row, piece_chars);
