Detailní popis

Enhanced Castling System pro ESP32 Sachovou Sachovnici.

Tento modul poskytuje komplexni, robustni system pro rosadu s:

  • Centralizovanou spravou stavu
  • Pokrocilym LED navodem a indikaci chyb
  • Inteligentnim error recovery
  • Timeout handlingem
  • Vizualnim navodem pro hrace
Autor
ESP32 Chess Team
Datum
2024

Enhanced Castling System poskytuje pokrocily system pro spravu rosady. Pomaha hracum s provedenim rosady pomoci LED navodu, detekuje chyby a umoznuje automatickou napravu.
