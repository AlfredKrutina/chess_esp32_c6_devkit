Problém: zsh: command not found: flutter
----------------------------------------
Flutter NENÍ Python balíček — příkaz „pip install flutter“ je špatně.

macOS (doporučeno, máš Homebrew v /opt/homebrew):
  export HOMEBREW_NO_AUTO_UPDATE=1
  brew install --cask flutter

Pak ověř (nový terminál nebo znovu načti shell):
  which flutter
  flutter doctor

Ručně (oficiální SDK):
  https://docs.flutter.dev/get-started/install/macos
  Po rozbalení přidej do ~/.zshrc řádek:
  export PATH="$PATH:/cesta/k/flutter/bin"

První spuštění projektu:
  cd .../free_chess_v1_mqtt_HA/flutter_czechmate
  flutter pub get
  flutter create .          ← doplní android/ a ios/, pokud chybí
  flutter run --dart-define=STAGING=true

Důležité: pokud už v promptu vidíš „... flutter_czechmate %“, jsi UVNITŘ
složky projektu — NEPŘIDÁVEJ znovu „cd flutter_czechmate“ (ta podsložka
neexistuje). Stačí:
  flutter pub get && flutter run --dart-define=STAGING=true

Poznámka: brew install může trvat dlouho (stahování SDK).
