Kontrola verze mobilní aplikace

Aplikace CzechMate při otevření nastavení stáhne soubor app_update.json z kořene tohoto webu a porovná semver verzi instalace s polem latest_version.

Pro klienta je zdrojem pravdy vždy JSON. Tato stránka popisuje schéma polí.

Umístění

app_update.json — stejný obsah jako v repozitáři v gh-pages-ready/app_update.json.

Schéma

  • latest_version (řetězec, povinné) — nejnovější vydaná verze. Starší instalace zobrazí nabídku aktualizace.
  • min_supported_version (řetězec, volitelné) — pod touto verzí se zobrazí výraznější výzva k aktualizaci.
  • release_page_url (řetězec URL, volitelné) — cíl tlačítka ke stažení. Při absenci se použije výchozí adresa z kódu aplikace.

Příklad

{
  "latest_version": "1.7.4",
  "min_supported_version": "1.6.0",
  "release_page_url": "https://alfredkrutina.github.io/chess_esp32_c6_devkit/downloads.html"
}

Po vydání nové verze aplikace se hodnoty v JSON srovnají s flutter_czechmate/pubspec.yaml, změna se commitne do repozitáře a nasadí přes GitHub Actions na větev gh-pages.
