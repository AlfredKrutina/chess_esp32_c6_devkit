Detailní popis

Implementace LED State Manageru.

Tento modul spravuje stav LED systemu a poskytuje pokrocile funkce pro ovladani LED. Umožnuje vrstveni efektu a spravu priority zobrazeni.

Autor
Alfred Krutina
Verze
2.5.0
Datum
2025-09-06

LED State Manager je centralni system pro spravu stavu LED. Umožnuje vrstveni ruznych efektu, spravu priority a pokrocile funkce pro ovladani LED systemu.
