◆ disable_all_colors()

static void disable_all_colors ( void  )
inlinestatic

Disable all ANSI color output.

Call this function if the terminal doesn't support ANSI codes. Macros will still work but produce plain text output.

Note: Current implementation doesn't support runtime disable. This is a placeholder for future enhancement.
