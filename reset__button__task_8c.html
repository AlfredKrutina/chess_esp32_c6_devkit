Detailní popis

Reset button task: ovladani reset tlacitka pro restart hry.

Tento modul implementuje:

  • Inicializaci reset button tasku a FreeRTOS komponent
  • Zpracovani reset tlacitka pro restart hry
  • Simulacni rezim bez hardware (pro development)
  • Integration s game taskem pro reset
  • Jedno tlacitko pro reset cele hry
Autor
Alfred Krutina
Verze
2.0
Datum
2025-08-16

Tento task zpracovava reset tlacitko pro restart hry. Kdyz hrac stiskne reset tlacitko, hra se restartuje do vychoziho stavu. Task detekuje stisknuti tlacitka a posila prikaz do game tasku.
