◆ uart_task_wdt_reset_safe()

esp_err_t uart_task_wdt_reset_safe ( void  )

Bezpecny reset WDT s logovanim WARNING misto ERROR pro ESP_ERR_NOT_FOUND.

Tato funkce bezpecne resetuje Task Watchdog Timer. Pokud task neni jeste registrovany (coz je normalni behem startupu), loguje se WARNING misto ERROR.

Návratová hodnota
ESP_OK pokud uspesne, ESP_ERR_NOT_FOUND pokud task neni registrovany (WARNING pouze)

Funkce je pouzivana pro bezpecny reset watchdog timeru behem UART operaci. Zabranuje chybam pri startupu kdy task jeste neni registrovany.
