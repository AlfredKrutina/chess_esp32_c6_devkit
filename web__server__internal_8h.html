◆ web_server_task_wdt_reset_safe()

esp_err_t web_server_task_wdt_reset_safe ( void  )

Reset TWDT jen pokud aktuální úloha je web_server_task.

build_snapshot_json a mutexové čekání se volají i z workerů httpd (GET snapshot) – ty nejsou v TWDT; esp_task_wdt_reset() pak vrací ESP_ERR_NOT_FOUND a komponenta task_wdt spamuje sériovku ERROR řádky.

Vrací ESP_OK i při přeskočení (cizí úloha nebo NULL handle).
